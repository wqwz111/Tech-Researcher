📋 Executive Summary

将 OpenClaw 用于团队或企业场景，需要解决三个核心问题：安全隔离（防止 Agent 越权访问）、成本控制（不同任务用不同模型）、运维管理（监控、诊断、会话生命周期）。本报告提供完整的解决方案，包括 Docker 沙箱配置、三层防御模型、模型分层策略、并发控制，以及一个 5 人团队的完整生产配置示例。

📦 Docker 沙箱配置

Per-Agent 沙箱

{
  agents: {
    list: [
      {
        id: "personal",
        workspace: "~/.openclaw/workspace-personal",
        sandbox: { mode: "off" }  // 完全信任，无沙箱
      },
      {
        id: "family",
        workspace: "~/.openclaw/workspace-family",
        sandbox: {
          mode: "all",         // 始终沙箱化
          scope: "agent",      // 每个 Agent 一个容器
          docker: {
            setupCommand: "apt-get update && apt-get install -y git curl",
            memoryLimit: "512m",
            networkMode: "bridge"
          }
        },
        tools: {
          allow: ["exec", "read", "sessions_spawn"],
          deny: ["write", "edit", "browser", "cron", "apply_patch"]
        }
      }
    ]
  }
}

沙箱模式

mode行为安全级别
off不使用沙箱，直接在 Host 运行⚠️ 仅限完全信任的 Agent
all所有操作都在沙箱内🔒🔒🔒 最高

沙箱 scope

scope行为资源开销
agent同一 Agent 的所有会话共享一个容器低
session每个会话独立容器高（但隔离更好）

🔧 工具权限分级

按风险级别分类

风险工具说明
🟢 低read, web_search, session_status只读操作
🟡 中write, edit, exec修改文件 / 执行命令
🔴 高browser, cron, nodes, gateway外部交互 / 系统级操作

配置示例

{
  tools: {
    subagents: {
      tools: {
        deny: ["gateway", "cron"],      // 子智能体禁止系统操作
        // allow: ["read", "exec"]       // 如设置，变为白名单模式
      }
    }
  }
}

⚡ 并发控制

{
  agents: {
    defaults: {
      subagents: {
        maxConcurrent: 8,          // 全局并发上限
        maxChildrenPerAgent: 5,    // 每个 Agent 最大活跃子任务
        maxSpawnDepth: 2,          // 最大嵌套深度
        archiveAfterMinutes: 60,   // 完成后自动归档时间
      }
    }
  },
  acp: {
    maxConcurrentSessions: 8       // ACP 会话并发上限
  }
}

  agents: {
    list: [
      { id: "ceo",    workspace: "~/.openclaw/workspace-ceo",    model: "anthropic/claude-opus-4-6" },
      { id: "dev",    workspace: "~/.openclaw/workspace-dev",    model: "anthropic/claude-sonnet-4-5" },
      { id: "design", workspace: "~/.openclaw/workspace-design", model: "anthropic/claude-sonnet-4-5" },
      { id: "pm",     workspace: "~/.openclaw/workspace-pm",     model: "anthropic/claude-sonnet-4-5" },
      { id: "ops",    workspace: "~/.openclaw/workspace-ops",    model: "anthropic/claude-sonnet-4-5",
        sandbox: { mode: "all", scope: "session" } }
    ],
    defaults: {
      subagents: {
        model: "anthropic/claude-sonnet-4-5",
        maxConcurrent: 16,
        maxChildrenPerAgent: 5,
        runTimeoutSeconds: 1800
      }
    }
  },
  bindings: [
    { agentId: "ceo", match: { channel: "telegram", peer: { kind: "direct", id: "tg:CEO_CHAT_ID" } } },
    { agentId: "dev", match: { channel: "discord", accountId: "dev-bot" } },
    { agentId: "dev", match: { channel: "telegram", accountId: "dev" } },
    { agentId: "design", match: { channel: "slack", accountId: "design-bot" } },
    { agentId: "pm", match: { channel: "whatsapp", accountId: "pm" } },
    { agentId: "ops", match: { channel: "discord", peer: { kind: "channel", id: "OPS_CHANNEL_ID" } } },
    { agentId: "ceo", match: { channel: "whatsapp" } }  // CEO 默认
  ],
  acp: {
    enabled: true,
    backend: "acpx",
    allowedAgents: ["codex", "claude"],
    maxConcurrentSessions: 4
  }
}
