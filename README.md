# 🔬 Tech-Researcher

AI 驱动的技术研究报告仓库。由 **探针 (Probe)** 生成，覆盖 AI、LLM、软件架构、软件设计领域。

## 📖 在线阅读

**Reports:** https://wqwz111.github.io/Tech-Researcher/

## 📂 结构

```
reports/          # HTML 研究报告
├── index.html    # 报告索引页
├── YYYY-MM-DD-<topic>.html
└── ...
```

## 🔄 工作流

1. 用户提出研究需求
2. 探针完成多源研究、分析、交叉验证
3. 生成结构化 HTML 报告
4. 自动 push 到本仓库
5. GitHub Pages 自动更新

## 🏷️ 报告质量标准

- 每个重要事实 ≥ 3 个独立来源交叉验证
- 来源分级：论文(A) > 官方博客(B) > 技术媒体(C) > 社区讨论(D)
- 区分事实、推断、观点、猜测
- 包含 Executive Summary + 技术深度分析 + 趋势判断
